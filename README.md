# Allan's Swap Bot

A custom cryptocurrency swap service for Allan's Server.

## Getting Started

To initiate a swap, use the tip.cc tipping system with the following syntax:

**Example:** To exchange $1 USDT for BUSD, send:
```
tip @Allan's Swap Bot $1 USDT w BUSD
```

You will receive the equivalent amount of BUSD in your tip.cc wallet.

## Commands

| Command | Description |
|---------|-------------|
| `/rates` | Display current exchange rates for all supported coins |
| `/swaphelp` | View detailed help and usage instructions |
| `/bals` | Show current bot wallet balances |

## Features

- Fast and reliable cryptocurrency swaps
- Real-time exchange rates
- Secure tip.cc wallet integration
- Simple command-line interface
